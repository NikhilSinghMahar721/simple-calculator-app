# simple-calculator-app
A reliable and fully responsive four-function calculator built using pure HTML, CSS, and vanilla JavaScript (no frameworks). This project focuses on demonstrating fundamental JavaScript logic for handling user input, managing state, and implementing arithmetic operations efficiently. The design uses CSS Grid for a flexible, mobile-first layout.
This is a clean, fully functional four-function calculator built from the ground up using pure HTML, CSS, and JavaScript. It serves as a foundational project to showcase core front-end development skills, focusing on robust logic and a responsive design without the use of external frameworks.

The calculator handles standard arithmetic operations (+, -, *, /), decimal input, and clear state management.
